# BOSSA AI OS

AI-powered restaurant operating system prototype for BOSSA Asado i Mar.

BOSSA AI OS is a portfolio-level AI systems and workflow prototype for turning market signals into clearer weekly decisions. It demonstrates practical product thinking across dashboards, prompts, decision memory, and operational planning.

---

## Live Preview

Run locally with any static file server.

Using Python:

```bash
cd src
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

Alternative using live-server:

```bash
npm install -g live-server
cd src
live-server
```

---

## 🎯 What is this?

BOSSA AI OS is a structured intelligence and decision system for restaurant operations.

It connects:
- market & competitor signals
- weekly executive briefing
- decision tracking
- action execution

The goal is simple:
Turn fragmented operations into a clear, repeatable system for thinking, deciding, and acting.

---

## What This Demonstrates

- AI-assisted operations workflow design
- dashboard/product prototyping
- decision logging and action tracking
- prompt system design
- structured documentation and roadmap planning

## ⚠️ Important
