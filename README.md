# BOSSA AI OS

AI-powered restaurant operating system for BOSSA Asado i Mar.

## Purpose
BOSSA AI OS is a structured intelligence and decision system for restaurant operations. It connects competitor monitoring, weekly analysis, executive review, and decision logging into one working system.

## Core layers
- Market & Competitor Intelligence
- Executive Command Center
- Weekly AI Briefing
- Decision Log
- AI Project Stack & Prompt Pack

## Repo structure
- `prompts/` = ChatGPT, Claude, Kimi, weekly brief, decision, and dashboard prompts
- `docs/` = architecture, workflows, operating rules, and project overview
- `notion/` = Notion page maps, command-center notes, and prompt-pack notes
- `src/` = future dashboard, API, and automation code
- `scripts/` = helper and setup scripts
- `archive/` = old drafts and temporary files
